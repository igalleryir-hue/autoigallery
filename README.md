# نمایشگاه iGallery - نسخه ارتقایافته ویترین خودرو

این نسخه با الهام از الگوهای موفق سایت‌های خرید و فروش خودرو بازطراحی شده است و برای نمایشگاه iGallery آماده اجرا روی Render است.

## امکانات اصلی

- ویترین لوکس مشکی با لوگوی مستطیلی iGallery
- پیام تبلیغاتی اصلی: خرید و فروش سریع خودرو
- فیلترهای حرفه‌ای‌تر خودرو:
  - جستجو بر اساس برند، مدل و ویژگی‌ها
  - محدوده قیمت
  - برند
  - سال تولید
  - حداکثر کارکرد
  - صفر / کارکرده
  - گیربکس
  - وضعیت بدنه
  - مرتب‌سازی جدیدترین، ارزان‌ترین و گران‌ترین
- کارت خودرو حرفه‌ای با برچسب‌های اعتماد:
  - تایید شده
  - گزینه تمیز / مدل بالا / قیمت قابل بررسی
  - بررسی مدارک، امکان معاوضه، تماس سریع
- مسیر معامله امن چهارمرحله‌ای
- بخش قیمت‌گذاری منصفانه، معاوضه، فروش سریع و خرید مطمئن
- پنل مدیریت جداگانه
- پنل کاربران جداگانه
- ثبت آگهی توسط کاربران با تایید نهایی مدیر
- ورود و ثبت‌نام کاربران با کد تایید پیامکی OTP
- آماده برای اتصال به پنل پیامکی کاوه‌نگار

## اجرا روی سیستم شخصی

```bash
npm install
npm run init-db
npm start
```

سپس باز کنید:

```text
http://localhost:3000
```

## اجرا روی Render

Build Command:

```bash
npm install
```

Start Command:

```bash

```

```env

ADMIN_PASSWORD=رمز_مدیریت_خودت
JWT_SECRET=autoigallery-secret-123
NODE_ENV=production
SMS_PROVIDER=console

برای ارسال واقعی پیامک با کاوه‌نگار:

SMS_API_KEY=کلید_API_کاوه_نگار
SMS_TEMPLATE=verify
OTP_TTL_MINUTES=5
OTP_MAX_ATTEMPTS=5
```



## قیمت‌گذاری خودکار خودرو

در این نسخه بخش «قیمت‌گذاری خودکار» اضافه شده است:

- کاربر پس از ورود پیامکی می‌تواند مشخصات خودرو را وارد کند.
- سیستم به‌صورت خودکار بازه قیمت، قیمت پیشنهادی فروش و قیمت خرید فوری نمایشگاه را محاسبه می‌کند.
- الگوریتم از آگهی‌های تاییدشده داخل iGallery و نمونه‌های بازار که مدیر وارد می‌کند استفاده می‌کند.
- مدیر می‌تواند از پنل مدیریت، نمونه قیمت بازار از منابع مجاز مثل باما، دیوار، کارنامه، همراه‌مکانیک یا API رسمی وارد کند.
- درخواست‌های قیمت‌گذاری کاربران در پنل مدیریت ذخیره می‌شود و مدیر می‌تواند یادداشت کارشناسی یا قیمت نهایی را اضافه کند.

نکته مهم: برای اتصال واقعی به «تمام سایت‌های ایرانی» بهتر است API یا دسترسی رسمی/مجاز تهیه شود. این نسخه موتور قیمت‌گذاری، دیتابیس، فرم‌ها و پنل لازم را آماده کرده و با ورود نمونه قیمت یا اتصال API قابل تقویت است.

### مسیرهای جدید

پنل کاربر:

```text
/user.html#auto-valuation
```

پنل مدیریت درخواست‌های قیمت‌گذاری:

```text
/admin.html#pricing-admin
```

پنل مدیریت نمونه‌های بازار:

```text
/admin.html#market-samples-admin
```

### APIهای جدید

```text
POST /api/user/price-estimate
GET  /api/user/price-estimates
GET  /api/admin/price-requests
PATCH /api/admin/price-requests/:id
POST /api/admin/price-samples
GET  /api/admin/price-samples
```

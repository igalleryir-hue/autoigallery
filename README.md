# نمایشگاه iGallery - نسخه آماده آپلود GitHub

این نسخه برای آپلود مستقیم داخل GitHub و Deploy روی Render آماده شده است.

## تغییرات این نسخه

- طراحی صفحه اصلی مطابق طرح لوکس مشکی/طلایی
- لوگوی تمیز و بدون بک‌گراند شطرنجی
- حذف کامل پیامک یکبارمصرف OTP
- ورود کاربر با شماره موبایل
- ثبت‌نام ساده کاربر با نام، نام خانوادگی و شماره موبایل
- ثبت آگهی توسط کاربر با تایید نهایی مدیر
- پنل مدیریت جداگانه
- قیمت‌گذاری خودکار خودرو
- فرم قیمت‌گذاری هوشمند در صفحه اصلی
- فیلتر خودروها بر اساس برند، مدل، سال، قیمت و کارکرد
- نسخه آماده Render

## راه آپلود در GitHub

1. فایل ZIP را Extract کنید.
2. وارد پوشه Extract شده شوید.
3. محتویات داخل پوشه را در GitHub آپلود کنید؛ خود ZIP را آپلود نکنید.
4. در GitHub روی Commit changes بزنید.
5. Render خودش Deploy جدید را شروع می‌کند.

## فایل‌هایی که باید در ریشه GitHub دیده شوند

- public
- server.js
- package.json
- schema.sql
- init-db.js
- README.md
- .env.example

## تنظیمات لازم در Render

در Environment Variables این موارد را داشته باشید:

```env
ADMIN_PHONE=09123060749
ADMIN_PASSWORD=رمز_مدیریت_خودت
JWT_SECRET=autoigallery-secret-123
NODE_ENV=production
```

دیگر نیازی به `SMS_PROVIDER` نیست، چون پیامک یکبارمصرف حذف شده است.

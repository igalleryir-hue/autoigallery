# iGallery.ir - نمایشگاه آنلاین خودرو

نسخه سفارشی iGallery با طراحی الهام‌گرفته از صفحه لیست محصولات سایت‌های فروش خودرو، اما با مشخصات اختصاصی iGallery.

## امکانات

- صفحه اصلی/همه خودروها با چیدمان حرفه‌ای فروشگاهی
- هدر تماس، اینستاگرام و روبیکا
- جستجوی خودرو
- فیلتر قیمت از/تا
- فیلتر برند
- فیلتر خودرو صفر/کارکرده
- مرتب‌سازی جدیدترین، ارزان‌ترین و گران‌ترین
- کارت خودروهای حرفه‌ای
- پنجره جزئیات خودرو با مشخصات کامل و هشدار امنیتی
- دکمه تماس و کپی شماره
- پنل مدیریت جداگانه: `/admin.html`
- پنل کاربران جداگانه: `/user.html`
- ثبت‌نام کاربران با نام، نام خانوادگی و شماره موبایل
- ثبت آگهی توسط کاربران با وضعیت «در انتظار تایید»
- انتشار آگهی فقط بعد از تایید نهایی مدیر
- بخش «تایید آگهی‌ها» در پنل مدیریت
- ثبت خودرو مستقیم توسط مدیر با انتشار فوری
- دیتابیس SQLite


سپس در مرورگر باز کنید:

## ورود امن کاربران با کد پیامکی

در این نسخه، ثبت‌نام و ورود کاربران با کد یک‌بارمصرف انجام می‌شود:

1. کاربر شماره موبایل را وارد می‌کند.
2. سایت یک کد ۶ رقمی می‌سازد.
3. کد تا ۵ دقیقه معتبر است.
4. بعد از تایید کد، کاربر وارد پنل می‌شود.
5. اگر پنل پیامکی تنظیم نشده باشد، کد در Logs سرور نمایش داده می‌شود تا برای تست استفاده شود.

### تنظیمات لازم در Render

در بخش Environment Variables این موارد را اضافه کنید:

```env
SMS_PROVIDER=console
SMS_API_KEY=
SMS_TEMPLATE=verify
OTP_TTL_MINUTES=5
OTP_MAX_ATTEMPTS=5
```

برای تست اولیه، `SMS_PROVIDER=console` بگذارید. در این حالت کد ورود داخل Logs رندر نمایش داده می‌شود.

برای ارسال واقعی پیامک با کاوه‌نگار:

```env
SMS_PROVIDER=kavenegar
SMS_API_KEY=کلید_API_پنل_پیامکی
SMS_TEMPLATE=verify
```

در پنل پیامکی باید قالب/Template با نام `verify` بسازید که توکن کد را در پیامک قرار دهد.
